# planningPoker

Planning poker app repo

# prerequisites:

1. PipEnv instructions for windows -> https://www.pythontutorial.net/python-basics/install-pipenv-windows/
2. Install npm version - 6.14.11 (Node version v14.16.0) [you can check npm version using command "npm --version"]

# To run

Python :

1. open term1)inal
2. run command - "pipenv shell"
3. run command - "python run.py"

VueJs:

1. open terminal, don't kill python terminal
2. npm install
3. npm run serve
